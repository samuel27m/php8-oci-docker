# php8-oci-docker
Docker-compose with PHP 8 &amp; OCI Extensions, Apache 2.4, Mailcatcher

## Apache
**Version:** 2.4.38

## PHP
**Version:** 8.0.0

**Extensions enabled:** OCI & MySQL
#### If you need more PHP extensions, change the `php-apache/Dockerfile` file and restart the containers
